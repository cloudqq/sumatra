this is sumatra

  1. We instantiate the plugin:

            sumatra 'clickToAlert', ->
          class PluginHelper extends SumatraPlugin
            action: 'click'

  2. An alert dialog should have popped up. This means it was initialized.

            initialize: ->
          alert 'initialized'
          true

    Remember, you have to return true to make the plugin continue to bind events.

  3. Now, when you click here, you should get another alert which confirms the click. This is accomplished with the perform() event: 

            perform: (event) =>
          event.preventDefault()
          alert 'clicked'
    We bound it like so: jQuery -> $('a').clickToAlert()
